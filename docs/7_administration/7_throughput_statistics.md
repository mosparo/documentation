# Throughput Statistics

On this page, you can see a chart of all processed submissions across all projects in your mosparo installation. Additionally, you can see all delayed or blocked requests across all projects.

You can adjust the chart's date range using the dropdown in the top-right corner.

## Active delays and lockouts

At the bottom of the page, you can see the active delays and lockouts. A delay or a lockout occurs when a project uses the security features "Delay" or "IP lockout", and a user makes too many requests.

You can reset the active delays or lockouts by clicking the button below the numbers. If you do that, all blocked users can use the forms again (until they are blocked again if they make too many requests).
