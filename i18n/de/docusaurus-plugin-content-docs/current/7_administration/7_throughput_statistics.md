# Durchsatzstatistiken

Auf dieser Seite sehen Sie eine Übersicht aller verarbeiteten Einsendungen aus allen Projekten Ihrer mosparo-Installation. Ausserdem werden alle verzögerten oder blockierten Anfragen aus allen Projekten angezeigt.

Über das Dropdown-Menü in der oberen rechten Ecke können Sie den Datumsbereich der Diagramme anpassen.

## Aktive Verzögerungen und Sperren

Am Ende der Seite werden die aktuellen Verzögerungen und Sperren angezeigt. Eine Verzögerung oder Sperre tritt ein, wenn ein Projekt die Sicherheitsfunktionen "Anfrageverzögerung" oder "IP-Sperre" nutzt und ein Benutzer zu viele Anfragen stellt.

Sie können die aktuellen Verzögerungen oder Sperren zurücksetzen, indem Sie auf die Schaltfläche unterhalb der Zahlen klicken. In diesem Fall können alle gesperrten Benutzer die Formulare wieder nutzen (bis sie erneut gesperrt werden, falls sie zu viele Anfragen machen).
